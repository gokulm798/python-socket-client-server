# Client-Server Socket Programming (Python)

## 📌 Problem
Enabling communication between systems over a network is a fundamental requirement in distributed applications.

## 🎯 Objective
Develop a client-server application using Python sockets for real-time communication.

## 🛠 Tools
Python, Socket Programming (TCP)

## 🔍 Work Done
- Implemented server to listen for client connections  
- Built client to connect and send messages  
- Established two-way communication between client and server  

## 📈 Key Concepts
- Client-server architecture  
- TCP/IP communication  
- Socket creation, binding, and connection  

## 💡 Impact
Demonstrates core networking concepts used in real-world applications like web services and messaging systems.

## ▶️ How to Run
Run server first, then client in separate terminals.
